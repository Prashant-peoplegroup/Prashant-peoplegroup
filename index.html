Some random notes on trying hands on few random services viz Dropbox, Amaon AWS, etc.

Recently I came across one nice script written by Andrea Fabrizi for accessing Dropbox through command-line. Though its named as Dropbox Uploader, it can do so many things like

  • upload
  • download
  • delete
  • move
  • copy
  • mkdir
  • list
  • share
  • saveurl
  • info
  • unlink
